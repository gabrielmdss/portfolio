Projetos em Destaque

Alguns dos projetos que desenvolvi utilizando as mais modernas tecnologias

Migração de Sistema Legado

Projeto particular realizado para uma empresa, com foco na migração de um sistema legado em Java para uma arquitetura moderna utilizando Angular no frontend e Node.js no backend, incluindo a integração com um sistema de emissão de nota fiscal eletrônica (NF-e).

  • Angular
  • Node.js
  • TypeScript
  • Express.js
  • Integração NF-e

Extensão Mobile de ERP Legado

Aplicativo mobile multiplataforma desenvolvido com React Native e Expo como extensão de um sistema ERP legado desenvolvido em Centura. O app permite acesso remoto a funcionalidades essenciais do ERP, como visualização de dados, controle de pedidos e integração com o backend existente.

  • React Native
  • Expo
  • TypeScript
  • Integração com ERP em Centura
  • Context API

App Mobile Integrado ao Sistema Apolo

Aplicativo mobile multiplataforma desenvolvido com React Native e Expo para integração ao sistema Apolo. O app oferece funcionalidades para o gerenciamento de agendamentos, controle de fluxo de clientes e registro de atendimentos em clínicas, otimizando a rotina dos profissionais da saúde.

  • React Native
  • Expo
  • TypeScript
  • Integração com sistema Apolo
  • Gerenciamento de Agendamentos

Sistema de Folha de Pagamento

Desenvolvimento de um sistema web para gestão de folha de pagamento, utilizando JavaScript puro, jQuery e PHP. O sistema permite o cálculo de salários, descontos, benefícios e geração de relatórios mensais, atendendo às necessidades administrativas da empresa.

  • JavaScript (Vanilla)
  • jQuery
  • PHP
  • Folha de pagamento
  • Relatórios mensais
